# 🏦 Maghrebia Scoring Platform

This is a Flask-based web application developed for **Assurance Maghrebia**.  
The goal of this project is to build a **solvency scoring system** for new insurance clients.

---

## 🚀 Features

- 🔐 User login system (basic authentication)
- 🏠 Home dashboard
- 📊 Future scoring system for new insured clients
- 🧠 Backend built with Flask
- 🎨 Simple HTML/CSS frontend

---

## 🛠️ Tech Stack

- Python 3
- Flask
- HTML / CSS
- (Future: MySQL / XAMPP / Machine Learning model)
